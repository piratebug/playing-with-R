# Mapping Points with Milos
Here, I follow [Milos Agathon's](https://github.com/milos-agathon) YouTube tutorial
to map points and perform some basic geospatial analysis, only using R.

*Libraries*
- tidyverse
- stringr
- httr
- sf
- giscoR
- scales
- ggrepel
